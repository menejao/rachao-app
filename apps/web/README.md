# Dashboard visual

Estrutura visual do frontend:

- `src/components/layout`
  - shell autenticado
  - sidebar desktop
  - topbar
  - mobile nav
- `src/components/common`
  - headers
  - stat cards
  - empty/loading states
  - badges
  - wrappers responsivos
- `src/components/players`, `matches`, `teams`
  - componentes de dominio para telas principais
- `src/app`
  - paginas navegaveis por modulo

Direcao visual:

- dark mode padrao
- grafite como base
- verde campo como primaria
- amarelo para destaque/ranking
- cards grandes e arredondados
- mobile-first
